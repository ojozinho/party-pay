# Product Spec

Party Pay is an Android-first app for parties, bars, pagodes, clubs and one-counter events.

The customer flow lets a guest enter an event, browse a visual menu, add items to cart, simulate Pix payment, receive a pickup QR and track order status.

The bartender flow is operational only: paid queue, item details, preparation states, QR scanner simulation and delivery confirmation. Bartenders do not see revenue, commission, payout or admin settings.

The admin flow includes dashboard KPIs, menu management, order list, operators and simulated finance using a 5% Party Pay commission.
