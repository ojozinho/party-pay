# Design System

The prototype extracts the visual language from the supplied UI/UX files:

- Background: deep black `#0B0B0D`
- Surfaces: dark layered cards `#141418`, `#1C1C22`
- Accents: acid yellow `#FFD93D`, coral `#FF6B6B`, lavender `#C5B4FF`, success green `#B8FF4D`
- Layout: large rounded cards, poster product tiles, glowing sticky CTAs, high-contrast QR/ticket surfaces
- Type: bold display-style system text, small monospaced overlines, italic accent spans
- Motion: animated screen entry, card reveal and press feedback using Reanimated

Reference assets from the prompt pack are centralized in `src/assets`.
