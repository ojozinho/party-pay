# Prototype Scope

Included:

- Expo/React Native Android-first app
- pt-BR default language
- en-US secondary dictionary
- Customer, bartender and admin role entry
- Local mock products, event, orders and operators
- Simulated Pix, split, QR pickup and scanner validation
- Visual styling based on the supplied Party Pay UI/UX pack

Not included:

- Real backend
- Real Pix/payment processing
- Real Supabase integration
- Fiscal or stock integrations
- Production authentication/security
